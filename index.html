    一个基于Mkdocs+python+Github实现的文档网站¶

    安装教程¶

    如果你也想搭建一个,你可以建立一个自己的WIKI文档

    安装环境¶

    1. python版本: 3.6.6

    2. 依赖的python包:

    包名 模块名 版本
    mkdocs mkdocs 1.0.4
    mkdocs-material material 3.0.6
    Markdown markdown 3.0.1
    pymdown-extensions pymdownx 6.0

    颜色主题¶

    主色¶

    默认为 white 点击色块可更换主题的主色

    辅助色¶

    默认为 red 点击色块更换主题的辅助色
